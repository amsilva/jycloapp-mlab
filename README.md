# Jyclo App 🦎
##  tracking your fasting cycles 

Um PWA minimalista para ajudar praticantes de jejum intermitente a controlar seus checkpoints alimentares e acompanhar janelas de jejum.

Funcionalidades (v0)

    - Acompanhamento de janelas de jejum em andamento
    - Registro de checkpoints alimentares (quando o jejum é quebrado)
    - Contador de ciclos completados
    - Visualização simples do progresso

Tecnologias

    PWA (Progressive Web App)
    HTML/CSS/JavaScript puro (minimalista, lembram-se?)
    Armazenamento local (sem backend por enquanto)
